Module Demo

This is a demo page for the module builder workflow found at https://github.com/johnfmorton/vite-module-builder-w-ghpages-npm-template. The demo page imports a typescript file during development. It comese preconfigured with TailwindCSS.

The typescript file is compiled to javascript, a type definitions file, and a css file when hosted in a GitHub Pages page during production.
